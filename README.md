## Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/mariuszmmm/photo-catalog
   ```
2. **Navigate to the project directory:**
   ```bash
   cd photo-catalog
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the application:**
   ```bash
   npm start
   ```
